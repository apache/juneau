Project Modules

This project has declared the following modules:

Name Description
Apache Juneau Core Apache Juneau Core APIs
Apache Juneau REST Apache Juneau REST APIs
Apache Juneau Microservices Apache Juneau Microservice APIs
Apache Juneau Server Config Apache Juneau Configuration Server API
Apache Juneau Examples Apache Juneau Examples
Apache Juneau Unit Tests Apache Juneau Core API Unit Tests
Apache Juneau Beans Apache Juneau Serializable Beans
Apache Juneau Shaded Artifacts Parent module for shaded (uber) JAR artifacts that bundle multiple Juneau modules together. These artifacts are useful for build systems like Bazel that require explicit dependencies, or for simplified dependency management in projects that need multiple Juneau modules.
Apache Juneau Distribution Apache Juneau Distribution
Apache Juneau Documentation Docusaurus-based documentation for Apache Juneau. This module does not produce any Maven artifacts - it only serves to make the documentation folder importable into IDEs like Eclipse.
