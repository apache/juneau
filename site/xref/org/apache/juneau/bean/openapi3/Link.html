25   * The Link object represents a possible design-time link for a response.
26   *
27   * <p>
28   * The Link Object represents a possible design-time link for a response. The presence of a link does not guarantee 
29   * the caller's ability to successfully invoke it, rather it provides a known relationship and traversal mechanism 
30   * between responses and other operations.
31   *
32   * <h5 class='section'>OpenAPI Specification:</h5>
33   * <p>
34   * The Link Object is composed of the following fields:
35   * <ul class='spaced-list'>
36   * 	<li><c>operationRef</c> (string) - A relative or absolute reference to an OAS operation (mutually exclusive with <c>operationId</c>)
37   * 	<li><c>operationId</c> (string) - The name of an existing, resolvable OAS operation (mutually exclusive with <c>operationRef</c>)
38   * 	<li><c>parameters</c> (map of any) - A map representing parameters to pass to an operation as specified with <c>operationId</c> or identified via <c>operationRef</c>
39   * 	<li><c>requestBody</c> (any) - A literal value or expression to use as a request body when calling the target operation
40   * 	<li><c>description</c> (string) - A description of the link (CommonMark syntax may be used)
41   * 	<li><c>server</c> ({@link Server}) - A server object to be used by the target operation
42   * </ul>
43   *
44   * <h5 class='section'>Example:</h5>
45   * <p class='bjava'>
46   * 	<jc>// Create a link to another operation</jc>
47   * 	Link <jv>link</jv> = <jk>new</jk> Link()
48   * 		.setOperationId(<js>"getUserById"</js>)
49   * 		.setParameters(
50   * 			JsonMap.<jsm>of</jsm>(<js>"userId"</js>, <js>"$response.body#/id"</js>)
51   * 		)
52   * 		.setDescription(<js>"The id value returned in the response can be used as userId parameter in GET /users/{userId}"</js>);
53   * </p>
54   *
55   * <h5 class='section'>See Also:</h5><ul>
56   * 	<li class='link'><a class="doclink" href="https://spec.openapis.org/oas/v3.0.0#link-object">OpenAPI Specification &gt; Link Object</a>
57   * 	<li class='link'><a class="doclink" href="https://swagger.io/docs/specification/links/">OpenAPI Links</a>
58   * 	<li class='link'><a class="doclink" href="https://juneau.apache.org/docs/topics/JuneauBeanOpenApi3">juneau-bean-openapi-v3</a>
59   * </ul>
60   */
