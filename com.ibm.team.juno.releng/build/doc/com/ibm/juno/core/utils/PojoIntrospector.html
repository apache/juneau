com.ibm.juno.core.utils

Class PojoIntrospector

  • java.lang.Object
    • com.ibm.juno.core.utils.PojoIntrospector

  • public final class PojoIntrospector
extends Object
    Used to invoke methods on Objects using arguments in serialized form.

    Example:

    String s = "foobar"; String s2 = (String)new PojoIntrospector(s).invoke("substring(int,int)", "[3,6]"); // "bar"

    Author:
    James Bognar (jbognar@us.ibm.com)
