---
title: "BasicHtmlDocTemplate"
---

The [BasicHtmlDocTemplate](../apidocs/org/apache/juneau/html/BasicHtmlDocTemplate.html) class defines a default template for HTML documents created by [HtmlDocSerializer](../apidocs/org/apache/juneau/html/HtmlDocSerializer.html).
The HTML document created by this template consists of the following structure:

```xml
&lt;html&gt;
    &lt;head&gt;
        &lt;style type='text/css'&gt;
            CSS styles and links to stylesheets
        &lt;/style&gt;
    &lt;/head&gt;
    &lt;body&gt;
        &lt;header&gt;
            Page header
        &lt;/header&gt;
        &lt;nav&gt;
            Navigation links
        &lt;/nav&gt;
        &lt;aside&gt;
            Side-bar text
        &lt;/aside&gt;
        &lt;article&gt;
            Contents of serialized object
        &lt;/article&gt;
        &lt;footer&gt;
            Footer message
        &lt;/footer&gt;
    &lt;/body&gt;
&lt;/html&gt;

```
