The juneau-examples-rest-springboot class contains the same examples as juneau-examples-rest but also includes the following:

  • A starter class for invoking the examples using Spring Boot.
  • A resource resolver for resolving REST resources as injectable beans.
  • A POM that extends from spring-boot-starter-parent that allows you to build a fully-shaded executable jar.
  • Configuration files for deploying the microservice to Heroku.

The microservice app can be imported into Eclipse just like juneau-examples-rest.

The microservice can be started from the juneau-examples-rest-springboot.launch file. It will start up the microservice on port 5000 which you can then view through a browser.

The Spring Boot examples app has identical functionality to the normal examples app except for console command support.
