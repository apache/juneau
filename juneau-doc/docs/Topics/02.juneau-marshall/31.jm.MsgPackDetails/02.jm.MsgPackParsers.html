{title:'MessagePack Parsers', updated:'9.0.0'}

The {@link oaj.msgpack.MsgPackParser} class is used to parse MessagePack into POJOs.

The class hierarchy for the builder of this parser is:

  • {@link oaj.Context.Builder}
    • {@link oaj.BeanContextable.Builder}
      • {@link oaj.parser.Parser.Builder}
        • {@link oaj.parser.InputStreamParser.Builder}
          • {@link oaj.msgpack.MsgPackParser.Builder}

Refer to the builder javadocs for configurable settings.

The following pre-configured parsers are provided for convenience:

  • {@link oaj.msgpack.MsgPackParser}
    • {@link oaj.msgpack.MsgPackParser#DEFAULT DEFAULT}
