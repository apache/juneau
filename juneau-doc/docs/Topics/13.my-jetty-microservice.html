Starter Project Zip

my-jetty-microservice-{@property juneauVersion}.zip

The my-jetty-microservice.zip file is a predefined starter Eclipse project for developing REST microservices using the {@doc juneau-microservice-jetty juneau-microservice-jetty} package.

It includes a combination of the Juneau Core, Server, and Client APIs and all libraries needed to execute in a Java 1.8+ environment.
