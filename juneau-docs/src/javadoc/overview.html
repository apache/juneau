Apache Juneau™

Apache Juneau™ is a single cohesive Java ecosystem consisting of a comprehensive toolkit for marshalling POJOs to a wide variety of content types using a common framework, along with universal REST server and client APIs for creating Swagger-based self-documenting REST interfaces.

Key Features

  • Universal Serialization - Marshal POJOs to JSON, XML, HTML, URL-Encoding, UON, MessagePack, CSV, and more
  • REST Services - Create self-documenting REST interfaces with automatic Swagger/OpenAPI generation
  • Microservices - Build lightweight, standalone microservices with embedded Jetty
  • Configuration Management - Sophisticated configuration file API with variable resolution
  • Fluent Assertions - Powerful testing framework with fluent-style assertions
  • Zero Dependencies - Core marshalling requires no external dependencies

Project Modules

Core Modules

Module Description
juneau-marshall Core POJO marshalling support for JSON, XML, HTML, URL-encoding, UON, MessagePack, and CSV
juneau-marshall-rdf Extended marshalling support for RDF/XML, N3, N-Tuple, and Turtle formats
juneau-config Sophisticated configuration file API with variable resolution and listeners
juneau-assertions Fluent-style assertions API for comprehensive testing

REST Modules

Module Description
juneau-rest-server Universal REST server API for creating Swagger-based self-documenting REST interfaces
juneau-rest-client Universal REST client API for interacting with REST interfaces using POJOs
juneau-rest-mock Mock testing framework for REST services
juneau-rest-server-springboot Spring Boot integration for Juneau REST services

Microservice Modules

Module Description
juneau-microservice-core Core microservice functionality and configuration management
juneau-microservice-jetty Jetty-based microservice implementation for REST services

Quick Example

Marshalling Example

// A simple POJO
public class Person {
    public String name = "John Smith";
    public int age = 21;
}

// Serialize to various formats
Person person = new Person();

String json = Json.of(person);           // {"name":"John Smith","age":21}
String xml = Xml.of(person);             // <object><name>John Smith</name><age>21</age></object>
String html = Html.of(person);           // <table><tr><td>name</td><td>John Smith</td></tr>...</table>
String urlEncoding = UrlEncoding.of(person); // name=John+Smith&age=21

// Parse back from any format
Person p = Json.to(json, Person.class);

REST Server Example

@Rest(path="/petstore")
public class PetStoreResource extends BasicRestServlet {
    
    @RestGet(path="/pets/{id}")
    public Pet getPet(@Path int id) {
        return findPet(id);
    }
    
    @RestPost(path="/pets")
    public Ok addPet(@Content Pet pet) {
        addPet(pet);
        return Ok.OK;
    }
}

Key Benefits

  • KISS Principle - No auto-wiring, code generation, or dependency injection
  • Memory Efficient - Fast, safe parsing that's not susceptible to deserialization attacks
  • Minimal Dependencies - Core marshalling requires no external libraries
  • Comprehensive Testing - Exhaustively tested with thousands of unit tests
  • Self-Documenting - Automatic Swagger/OpenAPI documentation generation
  • Easy to Use - Simple APIs that are enjoyable to work with
  • Tiny Footprint - Approximately 1MB for core functionality

Getting Started

To get started with Apache Juneau:

  1. Add the appropriate Juneau modules to your project dependencies
  2. Create POJOs to represent your data structures
  3. Use the marshalling APIs to serialize/parse your data
  4. Build REST services using the @Rest annotations
  5. Deploy as a standalone microservice or in any Servlet 3.1+ container
