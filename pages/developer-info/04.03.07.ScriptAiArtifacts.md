These scripts produce and validate the machine-consumable AI knowledge artifacts for Juneau documentation.

## What They Generate

`generate-ai-artifacts.py` writes:

- `static/ai/manifest.json`
- `static/ai/juneau-knowledge.jsonl`
- `static/ai/taxonomy.json`

The output is deterministic to avoid noisy diffs and index churn.

## What They Validate

`check-ai-artifacts.py` verifies:

- generated artifact files are current (`--check` mode),
- required manifest shape exists,
- quickstart dependency version in `README.md` matches current release,
- Javadocs links in `docusaurus.config.ts` are internally consistent,
- current release has a canonical page in `pages/release-notes`.

## Usage

From the `juneau-docs` repository root:

```bash
python3 scripts/generate-ai-artifacts.py
python3 scripts/check-ai-artifacts.py
```

Or via npm scripts:

```bash
npm run build:ai-artifacts
npm run check:ai-artifacts
```

## Notes

- The generator consumes documentation pages and release notes plus selected API metadata signals from the sibling `juneau` repository.
- Keep summaries short and descriptive to support low-token retrieval workflows.
