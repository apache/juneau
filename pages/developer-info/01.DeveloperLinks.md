## Developer Links

Useful links for Apache Juneau developers:

- **[Jenkins](https://ci-builds.apache.org/job/Juneau/)** - Continuous integration and deployment
- **[Confluence](https://cwiki.apache.org/confluence/display/JUNEAU)** - Collaborative documentation and project management
- **[Confluence Blog](https://cwiki.apache.org/confluence/display/JUNEAU/Blog)** - Project updates and announcements
- **[Twitter](https://twitter.com/apachejuneau)** - Follow Apache Juneau on Twitter
- **[Slack](https://apachejuneau.slack.com)** - Real-time discussions and collaboration
- **[Apache Mailing Lists](https://juneau.apache.org/mail-lists.html)** - Project communication
  - **dev@juneau.apache.org** - Development discussions
  - **commits@juneau.apache.org** - Commit notifications
  - **users@juneau.apache.org** - User discussions
- **[Pony Mail](https://lists.apache.org/list.html?dev@juneau.apache.org)** - Email archive system
