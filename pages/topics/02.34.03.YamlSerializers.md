##### Usage

```java
// Serialize a POJO to YAML
String yaml = YamlSerializer.DEFAULT.serialize(myBean);

// Create a custom serializer
YamlSerializer s = YamlSerializer.create()
    .addBeanTypes()
    .trimNullProperties()
    .build();
String yaml = s.serialize(myBean);

// Use the Yaml marshaller for convenience
String yaml = Yaml.of(myBean);
```

##### Output Style

The YAML serializer produces block-style output by default.  Beans and maps are formatted with each property
on its own line, and collections/arrays use the YAML block sequence marker (`-`).

The `DEFAULT_READABLE` variant produces the same output as `DEFAULT` since YAML is inherently human-readable
through its indentation-based format.
