The [org.apache.juneau.commons.logging](/site/apidocs/org/apache/juneau/commons/logging/package-summary.html) package provides thin extensions over `java.util.logging` that add lazy message formatting and in-memory log capture for testing.

## Logger

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/logging/Logger.html" target="_blank">Logger</a></java-class>
An extended logger that wraps an underlying `java.util.logging.Logger` and adds convenience methods for logging formatted messages. Obtain one via the static factory methods.

```java
Logger logger = Logger.getLogger(MyClass.class);
logger.info("Started");
logger.warning("Low disk space");
```

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/logging/LogRecord.html" target="_blank">LogRecord</a></java-class>
A `java.util.logging.LogRecord` subclass that supports lazy message formatting — the message string is only built (via `StringUtils.format(String, Object...)`) if the record is actually published.

## Test Support

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/logging/LogRecordCapture.html" target="_blank">LogRecordCapture</a></java-class>
A `Closeable` that captures emitted log records in memory for inspection during tests.

```java
Logger logger = Logger.getLogger(MyClass.class);
try (LogRecordCapture capture = logger.captureEvents()) {
    doSomethingThatLogs();
    // assert against capture's collected records
}
```

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/logging/LogRecordListener.html" target="_blank">LogRecordListener</a></java-interface>
An interface for receiving notifications when log records are logged; implemented by `LogRecordCapture`.
