The YAML parser supports:

- Block-style mappings and sequences (indentation-based).
- Flow-style mappings (`{key: value}`) and sequences (`[item1, item2]`).
- Plain, single-quoted, and double-quoted scalars.
- Literal block scalars (`|`) and folded block scalars (`>`).
- Comments (`# ...`).
- Null values (`null`, `~`).
- Booleans (`true`, `false`).
- Integer and floating-point numbers.
- Document markers (`---`, `...`).
- Bean type dictionaries via `_type` property.

##### Usage

```java
// Parse YAML into a POJO
MyBean bean = YamlParser.DEFAULT.parse(yaml, MyBean.class);

// Parse into generic types
JsonMap map = YamlParser.DEFAULT.parse(yaml, JsonMap.class);
JsonList list = YamlParser.DEFAULT.parse(yaml, JsonList.class);

// Parse with generic type parameters
List<MyBean> beans = YamlParser.DEFAULT.parse(yaml, List.class, MyBean.class);

// Use the Yaml marshaller for convenience
MyBean bean = Yaml.to(yaml, MyBean.class);
```
