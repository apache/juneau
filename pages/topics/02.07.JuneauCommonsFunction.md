The [org.apache.juneau.commons.function](/site/apidocs/org/apache/juneau/commons/function/package-summary.html) package extends the standard `java.util.function` toolkit with higher-arity functional interfaces, checked-exception–friendly variants, small immutable tuples, and lazy/nullable suppliers used throughout the framework.

## Higher-Arity Functional Interfaces

Two-to-five argument extensions of the standard `Consumer` and `Function` patterns.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Consumer2.html" target="_blank">Consumer2</a></java-interface>
An operation that accepts two arguments and returns no result. Higher-arity siblings: <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Consumer3.html" target="_blank">Consumer3</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Consumer4.html" target="_blank">Consumer4</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Consumer5.html" target="_blank">Consumer5</a></java-interface>.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Function2.html" target="_blank">Function2</a></java-interface>
A function that accepts two arguments and produces a result. Higher-arity siblings: <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Function3.html" target="_blank">Function3</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Function4.html" target="_blank">Function4</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Function5.html" target="_blank">Function5</a></java-interface>.

## Checked-Exception Variants

Functional interfaces whose functional method is allowed to throw checked exceptions — useful for passing exception-throwing code where the JDK interfaces would force a try/catch.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/Snippet.html" target="_blank">Snippet</a></java-interface>
Identical to `Runnable` but its `run()` method may throw a checked exception.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingConsumer.html" target="_blank">ThrowingConsumer</a></java-interface>
A `Consumer` whose `acceptThrows(T)` may throw a checked exception. Higher-arity siblings: <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingConsumer2.html" target="_blank">ThrowingConsumer2</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingConsumer3.html" target="_blank">ThrowingConsumer3</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingConsumer4.html" target="_blank">ThrowingConsumer4</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingConsumer5.html" target="_blank">ThrowingConsumer5</a></java-interface>.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingFunction.html" target="_blank">ThrowingFunction</a></java-interface>
A `Function` whose functional method may throw a checked exception. Higher-arity siblings: <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingFunction2.html" target="_blank">ThrowingFunction2</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingFunction3.html" target="_blank">ThrowingFunction3</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingFunction4.html" target="_blank">ThrowingFunction4</a></java-interface>, <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingFunction5.html" target="_blank">ThrowingFunction5</a></java-interface>.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/ThrowingSupplier.html" target="_blank">ThrowingSupplier</a></java-interface>
A `Supplier` whose `get()` may throw a checked exception.

## Lazy and Nullable Suppliers

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/NullableSupplier.html" target="_blank">NullableSupplier</a></java-interface>
A `Supplier` whose value may be `null`, with `Optional`-like null-safe query methods.

```java
NullableSupplier<String> s = NullableSupplier.ofNullable(computeOrNull());
```

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/Memoizer.html" target="_blank">Memoizer</a></java-class>
A thread-safe supplier that caches the result of its first call and can be reset.

```java
Memoizer<Config> config = new Memoizer<>(() -> loadExpensiveConfig());
Config c = config.get();   // computed once
config.reset();            // next get() recomputes
```

## Tuples

Immutable value tuples with content-based `equals()`/`hashCode()` — handy as composite map keys.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/Tuple1.html" target="_blank">Tuple1</a></java-class>
A single-value tuple. Wider tuples: <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/Tuple2.html" target="_blank">Tuple2</a></java-class>, <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/Tuple3.html" target="_blank">Tuple3</a></java-class>, <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/Tuple4.html" target="_blank">Tuple4</a></java-class>, <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/Tuple5.html" target="_blank">Tuple5</a></java-class>.

```java
Tuple2<String,Integer> key = Tuple2.of("foo", 42);
String a = key.getA();
Integer b = key.getB();
```

## Bean Streaming

Lifecycle-aware source/target interfaces that let the marshalling layer stream beans one at a time without buffering an entire document.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/BeanSupplier.html" target="_blank">BeanSupplier</a></java-interface>
A lazy, one-at-a-time producer used as a serializer source (extends `Iterable`).

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/BeanConsumer.html" target="_blank">BeanConsumer</a></java-interface>
A one-at-a-time consumer used as a parser target (extends `ThrowingConsumer`).

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/BeanChannel.html" target="_blank">BeanChannel</a></java-interface>
Combines `BeanSupplier` and `BeanConsumer` for round-trip use on the same property or class. <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/ListBeanChannel.html" target="_blank">ListBeanChannel</a></java-class> is a simple `ArrayList`-backed implementation.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/function/BeanFactory.html" target="_blank">BeanFactory</a></java-interface>
A universal factory interface used with `@Marshalled(factory=...)` as the single DI hook for framework-instantiated classes. <java-class><a href="/site/apidocs/org/apache/juneau/commons/function/BeanFactory.Void.html" target="_blank">BeanFactory.Void</a></java-class> is the sentinel default meaning "no factory specified".
