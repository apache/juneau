The [org.apache.juneau.commons.time](/site/apidocs/org/apache/juneau/commons/time/package-summary.html) package provides a precision-aware date/time type for granular time operations.

## Granular Date/Time

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/time/GranularZonedDateTime.html" target="_blank">GranularZonedDateTime</a></java-class>
Pairs a `java.time.ZonedDateTime` with a `ChronoField` precision identifier, enabling granular time operations such as rolling to the start or end of a period at a chosen precision (year, month, day, hour, …). It parses a wide range of string forms and can be built from a `Date`, a `ZonedDateTime`, or a string.

```java
// Parse a partial timestamp — precision is inferred from the input.
GranularZonedDateTime g = GranularZonedDateTime.of("2026-07");   // month precision

// Build from an existing ZonedDateTime at an explicit precision.
GranularZonedDateTime day = GranularZonedDateTime.of(ZonedDateTime.now(), ChronoField.DAY_OF_MONTH);
```
