Apache Juneau provides three minimal, clone-and-go **"Hello World" starter projects** — one per
deployment type. They are deliberately simpler than the [petstore showcase](/docs/topics/JuneauPetstore):
entry-point templates you can build and run in under five minutes, then graduate from.

Each starter is a standalone public GitHub repo with its own docs site:

| Starter | Deployment | Repo | Docs |
| --- | --- | --- | --- |
| Jetty | Embedded Jetty, runnable jar | [juneau-microservice-jetty-starter](https://github.com/jamesbognar/juneau-microservice-jetty-starter) | [Docs](https://jamesbognar.github.io/juneau-microservice-jetty-starter/) |
| Spring Boot | Spring Boot, runnable jar | [juneau-microservice-springboot-starter](https://github.com/jamesbognar/juneau-microservice-springboot-starter) | [Docs](https://jamesbognar.github.io/juneau-microservice-springboot-starter/) |
| Tomcat | WAR on Tomcat 10+ | [juneau-microservice-tomcat-starter](https://github.com/jamesbognar/juneau-microservice-tomcat-starter) | [Docs](https://jamesbognar.github.io/juneau-microservice-tomcat-starter/) |

Each starter demonstrates a lean, Juneau-native feature set: a POJO-returning resource (JSON/XML/HTML
content negotiation), the built-in HtmlDoc view, an auto-generated Swagger/OPTIONS page, `@Path`/`@Query`
binding, a YAML config file, and a `MockRestClient` example test exercising an `@Remote` typed client.
