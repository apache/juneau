Juneau supports converting arbitrary POJOs to and from TOML (Tom's Obvious Minimal Language) using native serializers and parsers implemented without any external dependencies.
TOML is a configuration file format that maps unambiguously to hash tables, widely used in projects like Cargo (Rust) and Python's `pyproject.toml`.
The <a href="/site/apidocs/org/apache/juneau/toml/TomlSerializer.html" target="_blank">TomlSerializer</a> converts POJOs directly to TOML format, and the <a href="/site/apidocs/org/apache/juneau/toml/TomlParser.html" target="_blank">TomlParser</a> creates POJOs directly from TOML without intermediate DOM objects.

##### Maven Dependency

TOML support is included in `juneau-marshall`:

```xml
<dependency>
    <groupId>org.apache.juneau</groupId>
    <artifactId>juneau-marshall</artifactId>
    <version>${juneau.version}</version>
</dependency>
```

##### Quick Start

```java
// Serialize a map to TOML
Map<String, Object> config = new LinkedHashMap<>();
config.put("name", "myapp");
config.put("port", 8080);

Map<String, Object> db = new LinkedHashMap<>();
db.put("host", "localhost");
db.put("port", 5432);
config.put("database", db);

String toml = Toml.of(config);
```

Produces:

```toml
name = "myapp"

[database]
host = "localhost"
port = 5432
```

##### Parsing

```java
// Parse TOML into a Map or JsonMap
JsonMap parsed = Toml.to(toml, JsonMap.class);
String name = parsed.getString("name");
JsonMap dbConfig = parsed.getMap("database");
String host = (String) dbConfig.get("host");
Long port = (Long) dbConfig.get("port");

// Parse into a bean
MyConfig config = Toml.to(toml, MyConfig.class);
```

##### Serialization

```java
// Use the default serializer
String toml = TomlSerializer.DEFAULT.serialize(someObject);

// Use the marshaller for convenience
String toml = Toml.of(someObject);
```

##### Supported TOML Features

The implementation conforms to the <a href="https://toml.io/en/v1.0.0" target="_blank">TOML v1.0.0 specification</a>:

- **Key-value pairs** - Strings, integers, floats, booleans, dates, and arrays
- **Tables** - `[section]` and nested `[parent.child]` tables
- **Inline tables** - `{key = "value"}` for simple nested structures
- **Arrays** - Homogeneous arrays of primitives, strings, and inline tables
- **Strings** - Basic (`"..."`), literal (`'...'`), and multi-line variants
- **Dates and times** - LocalDate, LocalTime, LocalDateTime, OffsetDateTime (ISO 8601)

##### Media Types

- **Produces**: `application/toml`
- **Accept**: `application/toml`, `text/toml`

##### REST Integration

```java
// Use BasicTomlConfig for TOML-only REST resources
@Rest
public class MyResource extends RestServlet implements BasicTomlConfig { ... }

// TOML is also included in BasicUniversalConfig
@Rest
public class MyResource extends RestServlet implements BasicUniversalConfig { ... }
```
