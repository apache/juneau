The RDF parsers convert RDF input (text or binary) into POJOs. The module provides character-based parsers for all text RDF formats and stream-based parsers for RDF/THRIFT and RDF/PROTO binary formats.

| Class | Format | Input |
|---|---|---|
| <a href="/site/apidocs/org/apache/juneau/jena/RdfXmlParser.html" target="_blank">RdfXmlParser</a> | RDF/XML | String, Reader |
| <a href="/site/apidocs/org/apache/juneau/jena/RdfJsonParser.html" target="_blank">RdfJsonParser</a> | RDF/JSON | String, Reader |
| <a href="/site/apidocs/org/apache/juneau/jena/NTripleParser.html" target="_blank">NTripleParser</a> | N-Triples | String, Reader |

| <a href="/site/apidocs/org/apache/juneau/jena/N3Parser.html" target="_blank">N3Parser</a> | N3 (Notation3) | String, Reader |

##### Usage

```java
// RDF/XML
MyBean bean = RdfXmlParser.DEFAULT.parse(rdfXmlString, MyBean.class);
MyBean bean2 = RdfXml.to(rdfXmlString, MyBean.class);

// Turtle, N-Triples, N3
MyBean fromTurtle = TurtleParser.DEFAULT.parse(turtleString, MyBean.class);
MyBean fromTurtle2 = Turtle.to(turtleString, MyBean.class);
MyBean fromNTriples = NTriple.to(nTriplesString, MyBean.class);
MyBean fromN3 = N3.to(n3String, MyBean.class);

// N-Quads, TriG, TriX
MyBean fromNQuads = NQuads.to(nQuadsString, MyBean.class);
MyBean fromTriG = TriG.to(triGString, MyBean.class);
MyBean fromTriX = TriX.to(triXString, MyBean.class);

// JSON-LD
MyBean fromJsonLd = JsonLdParser.DEFAULT.parse(jsonLdString, MyBean.class);
MyBean fromJsonLd2 = JsonLd.to(jsonLdString, MyBean.class);

// RDF/THRIFT and RDF/PROTO binary
MyBean fromThrift = RdfThrift.to(thriftBytes, MyBean.class);
MyBean fromProto = RdfProto.to(protoBytes, MyBean.class);

// Generic maps (specify key/value types)
Map<String,String> map = RdfThrift.to(bytes, Map.class, String.class, String.class);
```

##### Date/Time Parsing

RDF parsers use Juneau's built-in ISO 8601 support. Date/time and duration types in RDF literals are parsed via `Iso8601Utils.parse()` and converted to the target property type (`Calendar`, `Instant`, `ZonedDateTime`, `Duration`, etc.).

##### Custom Configuration

All RDF parsers support the standard Juneau parser builder options:

```java
RdfXmlParser p = RdfXmlParser.create()
    .swaps(DateSwap.class)
    .build();

TurtleParser p2 = TurtleParser.create()
    .swaps(DateSwap.class)
    .build();

RdfThriftParser p3 = RdfThriftParser.create()
    .swaps(MyCustomSwap.class)
    .build();
```

| N3Parser | `text/n3` |

| TriGParser | `application/trig` |
| TriXParser | `application/trix+xml` |

| RdfProtoParser | `application/vnd.apache.protobuf` |
