Request HTTP parts can also be retrieved programmatically through the following classes that can be passed in as
parameters or access through <a href="/site/apidocs/org/apache/juneau/rest/server/RestRequest.html" target="_blank">RestRequest</a> bean:

<tree>
<node-0><javac-class><a href="/site/apidocs/org/apache/juneau/rest/server/httppart/RequestHeaderList.html" target="_blank">RequestHeaderList</a></javac-class> <javac-class><a href="/site/apidocs/org/apache/juneau/rest/server/httppart/RequestQueryParamList.html" target="_blank">RequestQueryParamList</a></javac-class> <javac-class><a href="/site/apidocs/org/apache/juneau/rest/server/httppart/RequestFormParamList.html" target="_blank">RequestFormParamList</a></javac-class> <javac-class><a href="/site/apidocs/org/apache/juneau/rest/server/httppart/RequestPathParamList.html" target="_blank">RequestPathParamList</a></javac-class></node-0>
</tree>

:::tip Example
```java
@RestPost(...)
public Object myMethod(RequestHeaderList headers) {

    // Add a default value.
    headers.addDefault("ETag", DEFAULT_UUID);

    // Get a header value as a POJO.
    UUID etag = headers.get("ETag").as(UUID.class).get();

    // Get a header as a standard HTTP part.
    ContentType contentType = headers.get(ContentType.class).orElse(ContentType.TEXT_XML);
}
```
:::

Built in to these APIs are various convenience methods such as converting parts to different types or inline fluent
assertions:

:::tip Example
```java
// Inline fluent assertion and retrieval.
String foo = request
    .getHeader("Foo")
    .assertString().contains("bar")
    .get();
```
