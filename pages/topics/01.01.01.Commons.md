# Commons

The [juneau-commons](/docs/topics/JuneauCommonsBasics) module is the foundational layer of the Juneau ecosystem.
It has no framework dependencies — just Java — and provides the low-level utilities, collections, I/O helpers,
reflection tools, settings management, and dependency-injection support that all other Juneau modules build on top of.
Because every other Juneau module depends on it transitively, its classes are always on the classpath whenever you
use Juneau.

### String & General Utilities

[org.apache.juneau.commons.utils](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/utils/package-summary.html) contains the most frequently used utility classes in the codebase.

<java-class><a href="/site/apidocs/org/apache/juneau/commons/utils/StringUtils.html" target="_blank">StringUtils</a></java-class>
provides a large collection of static string-manipulation helpers:

```java
import static org.apache.juneau.commons.utils.StringUtils.*;

// Formatting (supports both MessageFormat and printf styles)
String s = format("Hello {0}, you have {1} items", "Alice", 5);
String t = format("Hello %s, you have %d items", "Alice", 5);

// Null/empty checks
isEmpty(null);       // true
isEmpty("");         // true
isNotEmpty("hi");    // true

// Abbreviation, padding, camel-casing
abbreviate("Hello World", 8);   // "Hello..."
camelToKebab("fooBarBaz");      // "foo-bar-baz"
```

<java-class><a href="/site/apidocs/org/apache/juneau/commons/utils/Utils.html" target="_blank">Utils</a></java-class>
is the general-purpose utility class imported statically across the codebase:

```java
import static org.apache.juneau.commons.utils.Utils.*;

// Concise collection literals
List<String>          list = l("a", "b", "c");
String[]              arr  = a("a", "b", "c");

// Null/empty tests
nn(obj);   // true if obj != null
n(obj);    // true if obj == null
ne(str);   // true if string is non-null and non-empty

// Stable hash across multiple values
int hash = h("key", 42, true);

// Memoized supplier (computed once, then cached)
NullableSupplier<String> lazy = memoize(() -> expensiveCall());
```

<java-class><a href="/site/apidocs/org/apache/juneau/commons/utils/DateUtils.html" target="_blank">DateUtils</a></java-class>
offers lightweight date/time formatting and parsing helpers without requiring a full date-parsing library.

### Collections

[org.apache.juneau.commons.collections](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/collections/package-summary.html) provides fluent wrappers, filtered views, multi-value maps, and
thread-safe caches on top of the standard Java Collections Framework.

```java
// Fluent collections — mutable wrappers with builder-style add/remove
FluentList<String> list = new FluentList<String>()
    .a("foo", "bar")
    .a("baz");

FluentMap<String, Integer> map = new FluentMap<String, Integer>()
    .a("one", 1)
    .a("two", 2);

// Multi-value map (one key → many values)
MultiMap<String, String> mm = new MultiMap<>();
mm.put("colors", "red");
mm.put("colors", "blue");
List<String> colors = mm.get("colors");  // ["red", "blue"]

// Thread-safe, size-bounded cache (LRU eviction)
Cache<String, MyBean> cache = Cache.of(1000);
MyBean bean = cache.get("key", k -> loadBean(k));

// Filtered/reversed views
FilteredList<String> view = new FilteredList<>(source, s -> s.startsWith("A"));
ReversedList<String> reversed = new ReversedList<>(source);
```

### I/O Utilities

[org.apache.juneau.commons.io](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/io/package-summary.html) provides file and stream helpers that complement the standard `java.io` API.

```java
// LocalFile — a rich File wrapper
LocalFile f = LocalFile.of("config/app.properties");
String text    = f.readText();
byte[] bytes   = f.readBytes();
f.writeText("updated content");

// LocalDir — directory navigation and listing
LocalDir dir = LocalDir.of("src/main/resources");
List<LocalFile> files = dir.files("*.json");

// Charset-safe file readers/writers
Reader r = FileReaderBuilder.of("data.txt").charset("UTF-8").build();

// Decorators
InputStream noClose = new NoCloseOutputStream(out);  // ignores close()
```

### Reflection Utilities

[org.apache.juneau.commons.reflect](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/reflect/package-summary.html) provides a rich, cached wrapper layer over the standard Java reflection API.

```java
// ClassInfo — cached wrapper for Class<?>
ClassInfo ci = ClassInfo.of(MyBean.class);
MethodInfo  mi = ci.getMethod("setName", String.class);
FieldInfo   fi = ci.getField("value");

// Annotation traversal across class hierarchies
ci.getAnnotations(MyAnnotation.class).forEach(a -> { /* ... */ });

// Check assignability, interfaces, etc.
ci.isAssignableTo(Serializable.class);
ci.isPublic();
ci.isAbstract();
```

### Settings & Property Sources

[org.apache.juneau.commons.settings](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/settings/package-summary.html) provides layered property lookup with support for system properties,
environment variables, `.env` files, manifest entries, command-line arguments, and custom sources.

```java
// Look up a property with automatic type conversion
StringSetting jvmVendor = Settings.get().get("java.vendor");
String vendor = jvmVendor.get();

Setting<Integer> timeout = Settings.get().get("app.timeout").asInteger();
int ms = timeout.orElse(5000);

// Thread-local override (handy in tests)
Settings.get().setLocal("app.timeout", "999");
// ... exercise the code under test ...
Settings.get().clearLocal();

// Composable property store: args → env → system props → defaults
PropertyStore store = PropertyStore.of(
    new ArgsPropertySource(argv),
    new SystemEnvPropertySource(),
    new SystemPropertyPropertySource()
);
String dbUrl = store.get("DB_URL").orElse("jdbc:h2:mem:");
```

### Dependency Injection

[org.apache.juneau.commons.inject](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/inject/package-summary.html) is a lightweight, JSR-330–compatible dependency-injection framework
used internally by Juneau's REST server, microservice, and config modules.
It supports `@Inject`, `@Named`, `@Singleton`, `@Primary`, `@Order`, `@Bean`, `@Configuration`,
`@Conditional`, `@PostConstruct`, and `@PreDestroy` without compile-time dependencies on
`jakarta.inject-api` or `jakarta.annotation-api`.

```java
// Build a bean store and add beans directly
BeanStore store = BeanStore.create()
    .bean(DataSource.class, myDataSource)
    .bean(String.class, "jdbc:h2:mem:", "dbUrl")
    .build();

// Retrieve beans by type (and optional name)
DataSource ds = store.getBean(DataSource.class).orElseThrow();

// Wire a class whose constructor is annotated with @Inject
MyService svc = store.createBean(MyService.class);
```

### Runtime Utilities

[org.apache.juneau.commons.runtime](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/runtime/package-summary.html) provides helpers for working with the JVM runtime environment.

<java-class><a href="/site/apidocs/org/apache/juneau/commons/runtime/Args.html" target="_blank">Args</a></java-class>
parses command-line arguments into a structured, queryable object:

```java
Args args = new Args(argv);

// Positional arguments
String first = args.get(0).orElse(null);

// Named options: -key value  or  --key value
String host = args.get("host").orElse("localhost");
int port = Integer.parseInt(args.get("port").orElse("8080"));

// Boolean flags: -verbose
boolean verbose = args.has("verbose");
```

<java-class><a href="/site/apidocs/org/apache/juneau/commons/runtime/ManifestFile.html" target="_blank">ManifestFile</a></java-class>
reads entries from `META-INF/MANIFEST.MF`:

```java
ManifestFile mf = ManifestFile.get();    // From system class-loader
String version = mf.get("Implementation-Version").orElse("dev");
```

### Logging

[org.apache.juneau.commons.logging](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/logging/package-summary.html) provides a thin, JUL-based logging abstraction used across the Juneau codebase.

<java-class><a href="/site/apidocs/org/apache/juneau/commons/logging/Logger.html" target="_blank">Logger</a></java-class>
wraps `java.util.logging.Logger` with printf-style formatting and fluent convenience methods.
<java-class><a href="/site/apidocs/org/apache/juneau/commons/logging/LogRecordCapture.html" target="_blank">LogRecordCapture</a></java-class>
lets tests intercept and assert on log output:

```java
try (LogRecordCapture capture = LogRecordCapture.install(MyService.class)) {
    myService.doSomething();
    List<LogRecord> records = capture.getRecords();
    // assert on records...
}
```

### Functional Utilities

[org.apache.juneau.commons.function](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/function/package-summary.html) extends `java.util.function` with multi-argument and throwing variants:

```java
// Functions with up to 5 arguments
Function2<String, Integer, String> f2 = (s, i) -> s.repeat(i);
Function3<String, Integer, Boolean, String> f3 = (s, i, b) -> b ? s.repeat(i) : s;

// Throwing variants (no need to catch checked exceptions inline)
ThrowingFunction<String, MyObj> parse = MyObj::parse;
ThrowingSupplier<Config> loader = Config::loadDefault;

// Memoizer — computed once, resettable
Memoizer<String> memo = memoizer(() -> expensiveComputation());
String val = memo.get();   // computed
memo.reset();              // force recomputation next call
```

### Simple Variable Language

[org.apache.juneau.commons.svl](https://juneau.apache.org/site/apidocs/current/index.html?org/apache/juneau/commons/svl/package-summary.html) implements the Simple Variable Language (SVL) engine — a lightweight
template language that resolves `$Var{...}` expressions at runtime.
SVL is used throughout Juneau's configuration, microservice, and REST layers to inject dynamic values
into strings.

See [Simple Variable Language](/docs/topics/SimpleVariableLanguageBasics).
