#### Contents/Features

The `juneau-marshall-rdf` library provides additional serializers and parsers for RDF.
These rely on the Apache Jena library to provide support for the following formats:

**Text-based formats:**
- RDF/XML
- RDF/XML-Abbrev
- RDF/JSON
- N-Triples (W3C)
- Turtle (W3C)
- N3 (Notation3)
- N-Quads
- TriG (Turtle with named graphs)
- TriX (XML-based)
- JSON-LD (W3C)

**Binary formats:**
- RDF/THRIFT (Apache Thrift encoding)
- RDF/PROTO (Protocol Buffers encoding)

See [RDF Basics](/docs/topics/RdfBasics) for full documentation.
