## Contents/Features

The **juneau-test** module is a lightweight, self-contained toolkit for writing readable, expressive
unit tests.  It depends only on JUnit 5 and the core Juneau commons — no heavyweight frameworks — and
is used throughout Juneau's own test suite as well as by the built-in assertion methods on the REST
client and server APIs.

It is organized into three complementary areas:

- **[Assertions Overview](AssertionsOverview)** — the fluent-style assertions API in the
  <a href="/site/apidocs/org/apache/juneau/test/assertions/package-summary.html" target="_blank">org.apache.juneau.test.assertions</a>
  package.  The <a href="/site/apidocs/org/apache/juneau/test/assertions/Assertions.html" target="_blank">Assertions</a>
  class provides typed, chainable assertion objects for strings, numbers, beans, collections, dates,
  throwables, and more.  It powers the inline assertions used by the REST APIs and works equally well
  standalone in unit tests.

- **[Bean-Centric Testing](BeanCentricTesting)** — the BCT framework in the
  <a href="/site/apidocs/org/apache/juneau/test/bct/package-summary.html" target="_blank">org.apache.juneau.test.bct</a>
  package.  The <a href="/site/apidocs/org/apache/juneau/test/bct/BctAssertions.html" target="_blank">BctAssertions</a>
  static methods (`assertBean`, `assertBeans`, `assertList`, `assertMapped`, …) collapse verbose,
  multi-line property extraction into single, intention-revealing assertions with a flexible nested
  property syntax.  Behavior is customizable via
  [Customization](Customization) (stringifiers, listifiers, swappers) and
  [Property Extractors](PropertyExtractors).

- **[JUnit Extensions](JuneauTestJUnitExtensions)** — a small JUnit 5 extension in the
  <a href="/site/apidocs/org/apache/juneau/test/junit/package-summary.html" target="_blank">org.apache.juneau.test.junit</a>
  package (formerly the standalone `juneau-junit5` module) for **test-time bean injection** —
  swapping production beans with test doubles inside a Juneau `BeanStore` for the duration of a test.
