The <a href="/site/apidocs/org/apache/juneau/marshall/Example.html" target="_blank">@Example</a> annotation in
`org.apache.juneau.marshall` identifies a canonical **example instance** of a POJO. Juneau surfaces these examples in
generated Swagger/OpenAPI documents and uses them wherever a representative value of a type is needed.

`@Example` can be placed on a static method, a static field, or a class, and can also be applied to `@Rest`-annotated
classes and `@RestOp`-annotated methods via
<a href="/site/apidocs/org/apache/juneau/marshall/ExampleApply.html" target="_blank">@ExampleApply</a>.

## On a Static Method

A `public static` method annotated with `@Example` that returns an instance of the enclosing type supplies the example.
The method takes either zero arguments or a single
<a href="/site/apidocs/org/apache/juneau/marshall/MarshallingSession.html" target="_blank">MarshallingSession</a> argument.

```java
public class Address {

    public String street;
    public String city;

    @Example
    public static Address example() {
        Address a = new Address();
        a.street = "123 Main St";
        a.city = "Springfield";
        return a;
    }
}
```

The example is then emitted when the type is serialized as a documentation example:

```java
String json = Json5.DEFAULT.of(Address.example());   // {street:'123 Main St',city:'Springfield'}
```

## `example` Method Convention

A `public static` method simply **named** `example` (with no arguments or one `MarshallingSession` argument) is picked
up as the example even without the annotation:

```java
public class Address {
    public static Address example() { ... }   // recognized by name
}
```

## On a Static Field

A `public static` field annotated with `@Example` supplies the example directly:

```java
public class Address {

    public String street;
    public String city;

    @Example
    public static final Address EXAMPLE = create("123 Main St", "Springfield");
}
```

## On a Class (JSON5 String Form)

Applied to a class, `@Example` takes a Lax-JSON (JSON5) string value describing the example — no separate factory
method or field is needed:

```java
@Example("{street:'123 Main St',city:'Springfield'}")
public class Address {
    public String street;
    public String city;
}
```

## The `@Marshalled(example)` Equivalent

The same JSON5 example can be declared through the
[@Marshalled(example)](/docs/topics/MarshalledAnnotation) attribute, which is useful when you are already using
`@Marshalled` for other marshalling settings:

```java
@Marshalled(example="{street:'123 Main St',city:'Springfield'}")
public class Address {
    public String street;
    public String city;
}
```

Both the class-level `@Example("...")` form and the `@Marshalled(example="...")` form set the same underlying POJO
example.

## Variable Resolution

The JSON5 string forms support
<a href="/site/apidocs/org/apache/juneau/marshall/Example.html" target="_blank">VarResolver.DEFAULT</a> substitution, so
an example value can reference configuration and other variables that are resolved when the example is read:

```java
@Marshalled(example="{apiKey:'$C{Demo/apiKey}'}")
public class Credentials {
    public String apiKey;
}
```

See [VarResolver.DEFAULT](/docs/topics/DefaultVarResolver) for the set of built-in variables.

## Precedence

When more than one example source is present for a type, Juneau resolves them in this order:

1. The JSON5 string form — `@Example("...")` on the class or `@Marshalled(example="...")`.
2. An example **method** — a static method annotated with `@Example` or named `example`.
3. An `@Example`-annotated static **field**.

The first source that is present wins.
