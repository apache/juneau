The [org.apache.juneau.commons.bean](/site/apidocs/org/apache/juneau/commons/bean/package-summary.html) package provides Juneau's bean-modeling layer — the annotations and introspection engine that describe how Java beans are recognized, read, written, and mapped, independently of any particular marshalling format. The marshalling modules build on this layer through a small set of SPI seams so the bean model stays decoupled from serialization concerns.

## Bean Modeling Annotations

Annotations that control how classes and their properties are introspected.

### <java-annotation><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanType.html" target="_blank">BeanType</a></java-annotation>
Applied to a class to control bean introspection and modeling — property inclusion/order, read-only/write-only properties, a custom `PropertyNamer`, a stop class, and more.

```java
@BeanType(properties="id,name,city", propertyNamer=PropertyNamerDLC.class)
public class Address {
    public String id, name, city;
}
```

### <java-annotation><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanProp.html" target="_blank">BeanProp</a></java-annotation>
Applied to fields, getters/setters, or constructor parameters to control property modeling — most commonly to override the property name via its `value()`/`name()`.

```java
public class Person {
    @BeanProp("full-name")
    public String getName() { ... }
}
```

### <java-annotation><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanCtor.html" target="_blank">BeanCtor</a></java-annotation>
Maps constructor arguments to property names on beans with read-only properties.

```java
public class Point {
    public final int x, y;

    @BeanCtor(properties="x,y")
    public Point(int x, int y) { this.x = x; this.y = y; }
}
```

### <java-annotation><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanIgnore.html" target="_blank">BeanIgnore</a></java-annotation>
Excludes a class, field, method, or constructor from bean introspection.

```java
public class Account {
    public String username;

    @BeanIgnore
    public String getPasswordHash() { ... }   // never treated as a bean property
}
```

### <java-annotation><a href="/site/apidocs/org/apache/juneau/commons/bean/Name.html" target="_blank">Name</a></java-annotation>
Specifies the parameter name used for bean property mapping (e.g. on a constructor parameter).

```java
public Point(@Name("x") int x, @Name("y") int y) { ... }
```

### <java-annotation><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanConfig.html" target="_blank">BeanConfig</a></java-annotation>
Specifies bean-modeling config properties (visibility rules, property-namer, not-bean packages/classes, etc.) on REST classes and methods.

## Bean Introspection Runtime

The runtime types that hold and expose bean metadata.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanMeta.html" target="_blank">BeanMeta</a></java-class>
Encapsulates all access to the properties of a bean class (like a souped-up `java.beans.BeanInfo`). Obtained via the static factory.

```java
BeanMeta<Address> bm = BeanMeta.of(Address.class);
```

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanInfo.html" target="_blank">BeanInfo</a></java-class>
SPI seam exposing the type-classification surface (is-bean, element/key/value types, instantiation) that the bean-runtime types need without coupling the bean-modeling layer to the marshalling layer.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanMap.html" target="_blank">BeanMap</a></java-class>
A `Map`-style wrapper around a Java bean, exposing its properties as map entries backed by `BeanMeta`.

```java
BeanMap<Address> m = BeanMap.of(new Address());
m.put("city", "Portland");
Object city = m.get("city");
```

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanMapEntry.html" target="_blank">BeanMapEntry</a></java-class>
Represents a single entry in a `BeanMap`.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanPropertyMeta.html" target="_blank">BeanPropertyMeta</a></java-class>
Contains metadata about a single bean property.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanPropertyValue.html" target="_blank">BeanPropertyValue</a></java-class>
Represents a simple bean property value together with the metadata associated with it.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanSession.html" target="_blank">BeanSession</a></java-interface>
The minimum runtime surface the bean-modeling layer needs from a marshalling session.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanConfigContext.html" target="_blank">BeanConfigContext</a></java-class>
Immutable runtime configuration for the bean-modeling layer, built via its nested `Builder`.

## Property Naming

Converts conventional bean property names into other conventions.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/PropertyNamer.html" target="_blank">PropertyNamer</a></java-interface>
Defines the API for converting a conventional bean property name to some other form.

```java
public interface PropertyNamer {
    String getPropertyName(String name);
}
```

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BasicPropertyNamer.html" target="_blank">BasicPropertyNamer</a></java-class>
The default namer — returns property names unchanged.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/PropertyNamerDLC.html" target="_blank">PropertyNamerDLC</a></java-class>
Converts property names to dashed-lower-case format (`fooBar` → `foo-bar`).

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/PropertyNamerDUCS.html" target="_blank">PropertyNamerDUCS</a></java-class>
Converts property names to dashed-upper-case-start format.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/PropertyNamerULC.html" target="_blank">PropertyNamerULC</a></java-class>
Converts property names to underscore-lower-case format (`fooBar` → `foo_bar`).

## Property Validation

Optional schema-driven validation of bean-property values.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/PropertyValidator.html" target="_blank">PropertyValidator</a></java-interface>
Validates a bean-property value against a schema, throwing `SchemaValidationException` on failure.

```java
void validate(Object value) throws SchemaValidationException;
```

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/PropertyValidatorFactory.html" target="_blank">PropertyValidatorFactory</a></java-interface>
Builds `PropertyValidator` instances from a JSON-Schema-shaped map of constraints.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/PropertyValidators.html" target="_blank">PropertyValidators</a></java-class>
Discovery helper that locates a `PropertyValidatorFactory` via `ServiceLoader`.

## Extension Points (SPI)

Seams that let the marshalling layer plug into the bean model without the bean-modeling package depending on marshalling.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanInterceptor.html" target="_blank">BeanInterceptor</a></java-class>
Intercepts bean property reads and writes, allowing values to be transformed during introspection. `BeanInterceptor.Void` is the non-existent (no-op) placeholder.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanFilter.html" target="_blank">BeanFilter</a></java-interface>
Exposes the per-class filter surface the bean-runtime types need, decoupled from the marshalling-side filter.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanMetaInitializer.html" target="_blank">BeanMetaInitializer</a></java-interface>
Exposes marshalling-aware `BeanMeta` construction hooks.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanPropertyPostProcessor.html" target="_blank">BeanPropertyPostProcessor</a></java-interface>
Applies marshalling-side post-processing to a `BeanPropertyMeta.Builder` after its validation succeeds.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanRegistryLookup.html" target="_blank">BeanRegistryLookup</a></java-interface>
Exposes the dictionary-lookup surface the bean-runtime types need, decoupled from the marshalling-side bean registry.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanTypeResolver.html" target="_blank">BeanTypeResolver</a></java-interface>
Exposes the type-resolution operations that the bean-property validation pass needs from the marshalling layer.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/bean/Delegate.html" target="_blank">Delegate</a></java-interface>
Marks an object that represents (wraps) another object.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/bean/BeanProxyInvocationHandler.html" target="_blank">BeanProxyInvocationHandler</a></java-class>
Provides an `InvocationHandler` for creating dynamic proxy instances of bean interfaces.
