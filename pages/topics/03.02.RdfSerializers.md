The RDF serializers convert POJOs into RDF representations. The module provides both character-based serializers (text formats) and stream-based serializers (RDF/THRIFT and RDF/PROTO binary formats).

##### Text-Based Serializers

| Class | Format | Output |
|---|---|---|
| <a href="/site/apidocs/org/apache/juneau/jena/RdfXmlSerializer.html" target="_blank">RdfXmlSerializer</a> | RDF/XML | String |
| <a href="/site/apidocs/org/apache/juneau/jena/RdfXmlAbbrevSerializer.html" target="_blank">RdfXmlAbbrevSerializer</a> | RDF/XML Abbreviated | String |
| <a href="/site/apidocs/org/apache/juneau/jena/RdfJsonSerializer.html" target="_blank">RdfJsonSerializer</a> | RDF/JSON | String |
| <a href="/site/apidocs/org/apache/juneau/jena/NTripleSerializer.html" target="_blank">NTripleSerializer</a> | N-Triples (W3C) | String |
| <a href="/site/apidocs/org/apache/juneau/jena/TurtleSerializer.html" target="_blank">TurtleSerializer</a> | Turtle (W3C) | String |
| <a href="/site/apidocs/org/apache/juneau/jena/N3Serializer.html" target="_blank">N3Serializer</a> | N3 (Notation3) | String |
| <a href="/site/apidocs/org/apache/juneau/jena/NQuadsSerializer.html" target="_blank">NQuadsSerializer</a> | N-Quads | String |
| <a href="/site/apidocs/org/apache/juneau/jena/TriGSerializer.html" target="_blank">TriGSerializer</a> | TriG | String |

##### Binary Serializers

| Class | Format | Output |
|---|---|---|
| <a href="/site/apidocs/org/apache/juneau/jena/RdfThriftSerializer.html" target="_blank">RdfThriftSerializer</a> | RDF/THRIFT | byte[] |
| <a href="/site/apidocs/org/apache/juneau/jena/RdfProtoSerializer.html" target="_blank">RdfProtoSerializer</a> | RDF/PROTO | byte[] |

##### Usage

```java
// RDF/XML
String xml = RdfXmlSerializer.DEFAULT.serialize(bean);
String xml2 = RdfXml.of(bean);

// Turtle, N-Triples, N3
String turtle = TurtleSerializer.DEFAULT.serialize(bean);
String turtle2 = Turtle.of(bean);
String nTriples = NTriple.of(bean);
String n3 = N3.of(bean);

// N-Quads, TriG, TriX (datasets / named graphs)
String nQuads = NQuads.of(bean);
String triG = TriG.of(bean);
String triX = TriX.of(bean);

// JSON-LD
String jsonLd = JsonLdSerializer.DEFAULT.serialize(bean);
String jsonLd2 = JsonLd.of(bean);

// RDF/THRIFT and RDF/PROTO binary
byte[] thrift = RdfThrift.of(bean);
byte[] proto = RdfProto.of(bean);
```

##### Custom Configuration

All RDF serializers extend the standard Juneau serializer hierarchy and support the same builder options (swaps, bean filters, etc.):

```java
RdfXmlSerializer s = RdfXmlSerializer.create()
    .swaps(DateSwap.class)
    .build();

TurtleSerializer s2 = TurtleSerializer.create()
    .swaps(DateSwap.class)
    .build();

RdfThriftSerializer s3 = RdfThriftSerializer.create()
    .swaps(MyCustomSwap.class)
    .build();
```

| TurtleSerializer | `text/turtle` |

| TriGSerializer | `application/trig` |
| TriXSerializer | `application/trix+xml` |
| JsonLdSerializer | `application/ld+json` |
| RdfThriftSerializer | `application/vnd.apache.thrift.binary` |
| RdfProtoSerializer | `application/vnd.apache.protobuf` |
