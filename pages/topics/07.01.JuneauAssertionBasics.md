### Maven Dependency

```xml
<dependency>
    <groupId>org.apache.juneau</groupId>
    <artifactId>juneau-test</artifactId>
    <version>${juneau.version}</version>
</dependency>
```

### Java Library

```text
juneau-test-0.0.0.jar
```

### OSGi Module

```text
org.apache.juneau.test_0.0.0.jar
```

### Contents/Features

The <a href="/site/apidocs/org/apache/juneau/test/assertions/package-summary.html" target="_blank">org.apache.juneau.test.assertions</a> package in Juneau is a powerful API for performing
fluent style assertions.
It is used to implement built-in assertion methods on both the server and client side APIs.
But it can also be used standalone for testing.
