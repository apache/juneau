The [org.apache.juneau.commons.conversion](/site/apidocs/org/apache/juneau/commons/conversion/package-summary.html) package provides a lightweight, pluggable type-conversion engine for converting objects between types without requiring a full marshalling context or session.

## Core API

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/conversion/Converter.html" target="_blank">Converter</a></java-interface>
The primary interface for converting objects between types. Use `BasicConverter.INSTANCE` for a ready-made singleton that handles a wide range of common conversions.

```java
Converter converter = BasicConverter.INSTANCE;
Integer i = converter.to("123", Integer.class);
```

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/conversion/BasicConverter.html" target="_blank">BasicConverter</a></java-class>
A concrete `CachingConverter` supporting common type conversions (numbers, booleans, dates, collections, arrays, and more) without a `MarshallingContext`. Exposes the shared `BasicConverter.INSTANCE` singleton.

```java
long[] a = BasicConverter.INSTANCE.to("1,2,3", long[].class);
```

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/conversion/CachingConverter.html" target="_blank">CachingConverter</a></java-class>
An abstract `Converter` that lazily discovers conversion functions on first use for a given input/output type pair and caches them for subsequent calls.

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/conversion/ConfigurableConverter.html" target="_blank">ConfigurableConverter</a></java-class>
A `BasicConverter` subclass that accepts one or more `ConversionFinder` strategies, consulted in order before the built-in conversions.

## Extension Points

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/conversion/Conversion.html" target="_blank">Conversion</a></java-interface>
A functional interface representing a single type conversion. The `args` parameter carries any type arguments needed for parameterized output types.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/conversion/ConversionFinder.html" target="_blank">ConversionFinder</a></java-interface>
A pluggable strategy for discovering `Conversion` functions between type pairs; passed to the `ConfigurableConverter` constructor.

### <java-interface><a href="/site/apidocs/org/apache/juneau/commons/conversion/ConverterSession.html" target="_blank">ConverterSession</a></java-interface>
Provides contextual objects to `Conversion` functions at call time.

## Exceptions

### <java-class><a href="/site/apidocs/org/apache/juneau/commons/conversion/InvalidConversionException.html" target="_blank">InvalidConversionException</a></java-class>
Unchecked exception thrown by `Converter.to(Object, Class)` when no conversion path exists between two types. Callers unsure whether a conversion is possible can test first rather than catch.
