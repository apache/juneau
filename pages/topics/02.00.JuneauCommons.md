The juneau-commons module is the foundational layer of the Juneau ecosystem.
It has no framework dependencies — just Java — and provides the low-level utilities, collections, I/O helpers,
reflection tools, settings management, and dependency-injection support that all other Juneau modules build on top of.
Because every other Juneau module depends on it transitively, its classes are always on the classpath whenever you
use Juneau.

## Contents/Features

The `juneau-commons` module provides common utilities and APIs used across the Juneau ecosystem.

**Key Packages:**

- **[Utils](JuneauCommonsUtils)** ([org.apache.juneau.commons.utils](/site/apidocs/org/apache/juneau/commons/utils/package-summary.html)): General utility classes for strings, collections, classes, dates, files, and more
- **[Collections](JuneauCommonsCollections)** ([org.apache.juneau.commons.collections](/site/apidocs/org/apache/juneau/commons/collections/package-summary.html)): Enhanced collection utilities including fluent collections, multi-maps, filtered collections, and caching utilities
- **[Lang](JuneauCommonsLang)** ([org.apache.juneau.commons.lang](/site/apidocs/org/apache/juneau/commons/lang/package-summary.html)): Language utilities including value wrappers, version management, string formatting, and state enums
- **[Reflection](JuneauCommonsReflection)** ([org.apache.juneau.commons.reflect](/site/apidocs/org/apache/juneau/commons/reflect/package-summary.html)): Comprehensive reflection utilities for working with classes, methods, fields, and annotations
- **[Beans](JuneauCommonsBeans)** ([org.apache.juneau.commons.bean](/site/apidocs/org/apache/juneau/commons/bean/package-summary.html)): The format-independent bean-modeling layer — annotations and the introspection engine describing how Java beans are recognized, read, written, and mapped
- **[Settings](JuneauCommonsSettings)** ([org.apache.juneau.commons.settings](/site/apidocs/org/apache/juneau/commons/settings/package-summary.html)): Thread-local and global settings management with support for functional sources and stores
- **[Function](JuneauCommonsFunction)** ([org.apache.juneau.commons.function](/site/apidocs/org/apache/juneau/commons/function/package-summary.html)): Functional programming utilities including higher-arity and checked-exception functional interfaces, tuples, and lazy/nullable suppliers
- **[Conversion](JuneauCommonsConversion)** ([org.apache.juneau.commons.conversion](/site/apidocs/org/apache/juneau/commons/conversion/package-summary.html)): A lightweight, pluggable type-conversion engine that converts objects between types without a full marshalling context
- **[I/O](JuneauCommonsIO)** ([org.apache.juneau.commons.io](/site/apidocs/org/apache/juneau/commons/io/package-summary.html)): File and stream utilities, console support, MIME type detection, and path builders
- **[Inject](JuneauCommonsInject)** ([org.apache.juneau.commons.inject](/site/apidocs/org/apache/juneau/commons/inject/package-summary.html)): JSR-330–aligned bean store + Spring-lite injection layer (`@Inject`, `@Named`, `@Qualifier`, `@Singleton`, `@Primary`, `@Order`, `@Bean`, `@Configuration`, `@Conditional`, `@PostConstruct`, `@PreDestroy`, and `Provider<T>`) without compile-time dependencies on `jakarta.inject-api` or `jakarta.annotation-api`
- **[Simple Variable Language](SimpleVariableLanguage)** ([org.apache.juneau.commons.svl](/site/apidocs/org/apache/juneau/commons/svl/package-summary.html)): Recursive `$var{key}` variable resolution with a built-in function catalog and reusable compiled templates
- **[Http](JuneauCommonsHttp)** ([org.apache.juneau.commons.http](/site/apidocs/org/apache/juneau/commons/http/package-summary.html)): Lightweight, dependency-free primitives for parsing and modeling HTTP header values — media types, content-negotiation ranges, and name/value parameters
- **[Http-Part](JuneauCommonsHttpPart)** ([org.apache.juneau.commons.httppart](/site/apidocs/org/apache/juneau/commons/httppart/package-summary.html)): Shared enumerations and exception describing HTTP parts (query, path, header, form-data, body) for OpenAPI-schema-based serialization and parsing
- **[Concurrent](JuneauCommonsConcurrent)** ([org.apache.juneau.commons.concurrent](/site/apidocs/org/apache/juneau/commons/concurrent/package-summary.html)): Concurrency utilities including auto-closeable simple locks, read-write locks, and nullable atomic references
- **[Logging](JuneauCommonsLogging)** ([org.apache.juneau.commons.logging](/site/apidocs/org/apache/juneau/commons/logging/package-summary.html)): Thin extensions over `java.util.logging` adding lazy message formatting and in-memory log capture for testing
- **[Runtime](JuneauCommonsRuntime)** ([org.apache.juneau.commons.runtime](/site/apidocs/org/apache/juneau/commons/runtime/package-summary.html)): Lean accessors for command-line arguments and JAR manifest files
- **[Time](JuneauCommonsTime)** ([org.apache.juneau.commons.time](/site/apidocs/org/apache/juneau/commons/time/package-summary.html)): Time utilities for granular, precision-aware date/time handling
