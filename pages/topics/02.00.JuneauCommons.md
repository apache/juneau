The juneau-commons module is the foundational layer of the Juneau ecosystem.
It has no framework dependencies — just Java — and provides the low-level utilities, collections, I/O helpers,
reflection tools, settings management, and dependency-injection support that all other Juneau modules build on top of.
Because every other Juneau module depends on it transitively, its classes are always on the classpath whenever you
use Juneau.

<DependencyInfo artifact="juneau-commons" bundle="org.apache.juneau.common" />

## Contents/Features

The `juneau-commons` module provides common utilities and APIs used across the Juneau ecosystem.

**Key Packages:**

- **Collections** ([org.apache.juneau.commons.collections](/site/apidocs/org/apache/juneau/commons/collections/package-summary.html)): Enhanced collection utilities including fluent collections, multi-maps, filtered collections, and caching utilities
- **I/O** ([org.apache.juneau.commons.io](/site/apidocs/org/apache/juneau/commons/io/package-summary.html)): File and stream utilities, console support, MIME type detection, and path builders
- **Inject** ([org.apache.juneau.commons.inject](/site/apidocs/org/apache/juneau/commons/inject/package-summary.html)): JSR-330–aligned bean store + Spring-lite injection layer (`@Inject`, `@Named`, `@Qualifier`, `@Singleton`, `@Primary`, `@Order`, `@Bean`, `@Configuration`, `@Conditional`, `@PostConstruct`, `@PreDestroy`, and `Provider<T>`) without compile-time dependencies on `jakarta.inject-api` or `jakarta.annotation-api`
- **Reflection** ([org.apache.juneau.commons.reflect](/site/apidocs/org/apache/juneau/commons/reflect/package-summary.html)): Comprehensive reflection utilities for working with classes, methods, fields, and annotations
- **Settings** ([org.apache.juneau.commons.settings](/site/apidocs/org/apache/juneau/commons/settings/package-summary.html)): Thread-local and global settings management with support for functional sources and stores
- **Utils** ([org.apache.juneau.commons.utils](/site/apidocs/org/apache/juneau/commons/utils/package-summary.html)): General utility classes for strings, collections, classes, dates, files, and more
- **Lang** ([org.apache.juneau.commons.lang](/site/apidocs/org/apache/juneau/commons/lang/package-summary.html)): Language utilities including value wrappers, version management, string formatting, and state enums
- **Function** ([org.apache.juneau.commons.function](/site/apidocs/org/apache/juneau/commons/function/package-summary.html)): Functional programming utilities including tuples, throwing functions, and resettable suppliers
- **Concurrent** ([org.apache.juneau.commons.concurrent](/site/apidocs/org/apache/juneau/commons/concurrent/package-summary.html)): Concurrency utilities including simple locks and read-write locks
- **Time** ([org.apache.juneau.commons.time](/site/apidocs/org/apache/juneau/commons/time/package-summary.html)): Time utilities for granular date/time handling and time providers
- **Annotation** ([org.apache.juneau.commons](/site/apidocs/org/apache/juneau/commons/package-summary.html)): Annotation utilities for working with annotation objects and groups
- **Conversion** ([org.apache.juneau.commons.conversion](/site/apidocs/org/apache/juneau/commons/conversion/package-summary.html)): Type conversion utilities

## String & General Utilities

[org.apache.juneau.commons.utils](/site/apidocs/org/apache/juneau/commons/utils/package-summary.html) contains the most frequently used utility classes in the codebase.

<java-class><a href="/site/apidocs/org/apache/juneau/commons/utils/StringUtils.html" target="_blank">StringUtils</a></java-class>
provides a large collection of static string-manipulation helpers:

```java
import static org.apache.juneau.commons.utils.StringUtils.*;

// Formatting (supports both MessageFormat and printf styles)
String s = format("Hello {0}, you have {1} items", "Alice", 5);
String t = format("Hello %s, you have %d items", "Alice", 5);

// Null/empty checks
isEmpty(null);       // true
isEmpty("");         // true
isNotEmpty("hi");    // true

// Abbreviation, padding, camel-casing
abbreviate("Hello World", 8);   // "Hello..."
camelToKebab("fooBarBaz");      // "foo-bar-baz"
```

<java-class><a href="/site/apidocs/org/apache/juneau/commons/utils/Utils.html" target="_blank">Utils</a></java-class>
is the general-purpose utility class imported statically across the codebase:

```java
import static org.apache.juneau.commons.utils.Utils.*;

// Concise collection literals
List<String>          list = l("a", "b", "c");
String[]              arr  = a("a", "b", "c");

// Null/empty tests
nn(obj);   // true if obj != null
n(obj);    // true if obj == null
ne(str);   // true if string is non-null and non-empty

// Stable hash across multiple values
int hash = h("key", 42, true);

// Memoized supplier (computed once, then cached)
NullableSupplier<String> lazy = memoize(() -> expensiveCall());
```

<java-class><a href="/site/apidocs/org/apache/juneau/commons/utils/DateUtils.html" target="_blank">DateUtils</a></java-class>
offers lightweight date/time formatting and parsing helpers without requiring a full date-parsing library.
