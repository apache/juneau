The attribute-oriented <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html" target="_blank">XmlFormat</a>
values move bean-property values between XML attributes and child elements:

- <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ATTR" target="_blank">XmlFormat.ATTR</a> promotes a single property to an attribute.
- <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ATTRS" target="_blank">XmlFormat.ATTRS</a> makes a whole bean render its properties as attributes, or lets a single `Map` property contribute arbitrary attributes.
- <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ELEMENT" target="_blank">XmlFormat.ELEMENT</a> forces a property back to a child element, overriding an `ATTRS` default applied to the class.

## Rendering a property as an attribute — ATTR

The <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ATTR" target="_blank">XmlFormat.ATTR</a> format can be applied to bean
properties to serialize them as XML attributes instead of elements.
Note that this only supports properties of simple types (e.g.
strings, numbers, booleans).

<table class="code-table">
<tr>
<th>Data type</th>
<th>JSON example</th>
<th>Without annotation</th>
<th>With annotation</th>
</tr>
<tr>
<td>

```java
class MyBean {
  @Xml(format=XmlFormat.ATTR)
  public String a;
}
```

</td>
<td>

```json
{
  a: 'foo'
}
```

</td>
<td>

```xml
<object>
  <a>foo</a>
</object>
```

</td>
<td>

```xml
<object a='foo'/>
```

</td>
</tr>
</table>

## Rendering all properties as attributes — ATTRS

The <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ATTRS" target="_blank">XmlFormat.ATTRS</a> format can be applied to bean
classes to force all bean properties to be serialized as XML attributes instead of child elements.

<table class="code-table">
<tr>
<th>Data type</th>
<th>JSON example</th>
<th>Without annotation</th>
<th>With annotation</th>
</tr>
<tr>
<td>

```java
@Xml(format=XmlFormat.ATTRS)
class MyBean {
  public String a;
  public int b;
}
```

</td>
<td>

```json
{
  a: 'foo',
  b: 123
}
```

</td>
<td>

```xml
<object>
  <a>foo</a>
  <b>123</b>
</object>
```

</td>
<td>

```xml
<object a='foo' b='123'/>
```

</td>
</tr>
</table>

## Overriding the class default — ELEMENT

The <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ELEMENT" target="_blank">XmlFormat.ELEMENT</a> format can be applied to
bean properties to override the <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ATTRS" target="_blank">XmlFormat.ATTRS</a>
format applied on the bean class.

<table class="code-table">
<tr>
<th>Data type</th>
<th>JSON example</th>
<th>Without annotation</th>
<th>With annotation</th>
</tr>
<tr>
<td>

```java
@Xml(format=XmlFormat.ATTRS)
class MyBean {
  public String a;
  @Xml(format=XmlFormat.ELEMENT)
  public int b;
}
```

</td>
<td>

```json
{
  a: 'foo',
  b: 123
}
```

</td>
<td>

```xml
<object>
  <a>foo</a>
  <b>123</b>
</object>
```

</td>
<td>

```xml
<object a='foo'>
  <b>123</b>
</object>
```

</td>
</tr>
</table>

## Arbitrary attributes from a Map — ATTRS

The <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ATTRS" target="_blank">XmlFormat.ATTRS</a> format can also be applied to a
single bean property of type `Map` to denote arbitrary XML attribute values on the element.
These can be mixed with other <a href="/site/apidocs/org/apache/juneau/marshall/xml/XmlFormat.html#ATTR" target="_blank">XmlFormat.ATTR</a>
annotated properties but there must not be an overlap in bean property names and map keys.

<table class="code-table">
<tr>
<th>Data type</th>
<th>JSON example</th>
<th>Without annotation</th>
<th>With annotation</th>
</tr>
<tr>
<td>

```java
class MyBean {
  @Xml(format=XmlFormat.ATTRS)
  public Map<String,Object> a;
  @Xml(format=XmlFormat.ATTR)
  public int b;
}
```

</td>
<td>

```json
{
  a: {
    k1: 'foo',
    k2: 123
  },
  b: 456
}
```

</td>
<td>

```xml
<object>
  <a>
    <k1>foo</k1>
    <k2 _type='number'>123</k2>
  </a>
  <b>456</b>
</object>
```

</td>
<td>

```xml
<object k1='foo' k2='123' b='456'/>
```

</td>
</tr>
</table>
