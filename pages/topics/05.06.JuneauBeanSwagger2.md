The `juneau-bean-swagger-v2` module provides Java beans for working with [Swagger 2.0](https://swagger.io/specification/v2/) documents and user interfaces.

## Overview

This module contains predefined POJOs for:

- Swagger 2.0 document structure
- API documentation generation
- Swagger UI rendering

The beans are simply POJOs with fluent-style `setX(...)` setters, so they can be serialized to JSON using any of the Juneau JSON serializers, or to other languages such as XML or HTML using the other serializers.

## Swagger Documents

The <a href="/site/apidocs/org/apache/juneau/bean/swagger/SwaggerBuilder.html" target="_blank">SwaggerBuilder</a> class is a utility class with predefined static factory methods that let you construct Swagger DTO beans in a minimal amount of code. Typically you static-import it and call the factory methods directly.

```java
import static org.apache.juneau.bean.swagger.SwaggerBuilder.*;
import java.net.URI;

Swagger swagger = swagger()
    .setSwagger("2.0")
    .setInfo(
        info("Swagger Petstore", "1.0.0")
            .setDescription("This is a sample server Petstore server.")
            .setTermsOfService("http://swagger.io/terms/")
            .setContact(
                contact().setEmail("apiteam@swagger.io")
            )
            .setLicense(
                license("Apache 2.0", URI.create("http://www.apache.org/licenses/LICENSE-2.0.html"))
            )
    )
    .setHost("petstore.swagger.io")
    .setBasePath("/v2")
    .setTags(
        tag("pet").setDescription("Everything about your Pets")
            .setExternalDocs(
                externalDocumentation("http://swagger.io", "Find out more")
            )
    )
    .addSchemes("http")
    .addPath("/pet", "post",
        operation()
            .setTags("pet")
            .setSummary("Add a new pet to the store")
            .setDescription("")
            .setOperationId("addPet")
            .addConsumes(MediaType.JSON, MediaType.XML)
            .addProduces(MediaType.JSON, MediaType.XML)
            .setParameters(
                parameterInfo("body", "body")
                    .setDescription("Pet object that needs to be added to the store")
                    .setRequired(true)
            )
            .addResponse("405", responseInfo("Invalid input"))
    );

// Serialize using the JSON serializer.
String swaggerJson = Json.of(swagger);

// Or just use toString() / asJson().
swaggerJson = swagger.asJson();
```

Produces:

Because these are standard Juneau beans, a Swagger document parses back from any supported format:

```java
import org.apache.juneau.marshall.json.*;

Swagger back = JsonParser.DEFAULT.parse(swaggerJson, Swagger.class);
```

## SwaggerUI

The <a href="/site/apidocs/org/apache/juneau/bean/swagger/ui/SwaggerUI.html" target="_blank">SwaggerUI</a> class is a DTO bean class for generating Swagger user interfaces from <a href="/site/apidocs/org/apache/juneau/bean/swagger/Swagger.html" target="_blank">Swagger</a> beans.

The class itself is nothing more than a POJO swap (`ObjectSwap<Swagger,Div>`) that swaps out <a href="/site/apidocs/org/apache/juneau/bean/swagger/Swagger.html" target="_blank">Swagger</a> beans with <a href="/site/apidocs/org/apache/juneau/bean/html5/Div.html" target="_blank">Div</a> elements when the requested media type is HTML.

```java
public class SwaggerUI extends ObjectSwap<Swagger,Div> {

    @Override
    public MediaType[] forMediaTypes() {
        // Only use this swap when the Accept type is HTML.
        return new MediaType[] {MediaType.HTML};
    }

    @Override
    public Div swap(MarshallingSession session, Swagger swagger) throws Exception {
        // Generate an HTML UI for the Swagger document.
        ...
    }
}
```

## Bean Classes

| Class | Purpose |
|---|---|
| `Swagger` | Top-level Swagger 2.0 document. |
| `Info` | API metadata (`title`, `version`, `description`, `contact`, `license`). |
| `Contact` / `License` | Contact and license sub-objects of `Info`. |
| `Tag` | Tag descriptor with optional external docs. |
| `Operation` | A single API operation on a path/method. |
| `ParameterInfo` | Operation parameter descriptor. |
| `ResponseInfo` | Operation response descriptor. |
| `SchemaInfo` / `Items` | Schema and array-item descriptors. |
| `HeaderInfo` | Response header descriptor. |
| `SecurityScheme` | Security-definition descriptor. |
| `ExternalDocumentation` | External-documentation reference. |
| `Xml` | XML serialization hints for a schema. |
| `SwaggerUI` | POJO swap that renders a `Swagger` document as an HTML UI. |

## Dependencies

- **Runtime:** `juneau-marshall` (and `juneau-bean-html5` for `SwaggerUI`)
- **External:** None
- **Part of:** `juneau-bean` module group
