The `org.apache.juneau.marshall.ini` package provides Juneau serializer and parser implementations for the INI
configuration file format.

The `IniSerializer` and `IniParser` classes integrate with the standard Juneau `Serializer` / `Parser` APIs and
support the `@Ini` and `@IniConfig` annotations for format-specific configuration.

:::note
Full documentation for the INI serializer/parser is coming soon. In the meantime, refer to the
[Marshallers overview](/docs/topics/Marshallers) for general serializer/parser configuration patterns,
and the Javadoc for `IniSerializer`, `IniParser`, and `IniConfig` for the available options.
:::
