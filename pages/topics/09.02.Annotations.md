The <a href="/site/apidocs/org/apache/juneau/http/package-summary.html" target="_blank">org.apache.juneau.http</a> package contains annotations for defining both
server and client side APIs.

The server-side APIs also use it for producing auto-generated Swagger documentation through the REST API itself.
